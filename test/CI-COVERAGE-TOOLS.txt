# CI-COVERAGE-TOOLS.txt — scripts under test/ that are TOOLS, not gates.
#
# diff_compiler_ci_shard_coverage.sh enumerates EVERY *.sh under test/ and test/wasm/ and
# demands that each one be either:
#   (a) run in CI (matched by a shard `pattern:`, or named literally by a job step), or
#   (b) listed in CI-COVERAGE-EXCEPTIONS.txt with a reason (a LEDGER — printed every run), or
#   (c) listed HERE, because it is not a gate at all.
#
# WHY THE GATE ENUMERATES EVERYTHING NOW. It used to enumerate only the globs it already
# knew about (`test/diff_compiler_*.sh`, later plus wasm/bootstrap). So it certified
# coverage while a dozen REAL GATES — cross_project_*, diff_native_cli, the effect_*_domain
# family, selfcompile_emit, manifest_emit — had never run in CI at all. A completeness check
# that picks its own scope will always certify itself complete. The only way out is to make
# the scope EVERYTHING and force every script to be classified exactly once.
#
# A script in this file makes a claim: "running me proves nothing about the compiler."
# If that is false, it belongs in a shard, not here.
#
# Format:  <stem>  <what it is>

# ── build / bootstrap ──
build_native_medaka   builds ./medaka (warm 2-stage). Invoked BY the build, not a check.
bootstrap_from_seed   cold-bootstraps from the seed. Run by the `seed-health` CI job by name.
build_oracles         builds test/bin probe binaries. The gates' dependency, not a gate.
build_wasm_oracle     builds the wasm probe binaries.
build_wasm_cmd        wasm build helper.
build_cmd             build helper.
assemble_check_main   probe assembly helper.
refresh_seed          re-mints compiler/seed/emitter.ll.gz. A maintenance action.

# ── harness / driver ──
run_gates             THE gate runner. Running it IS running the gates.
preflight             the agent inner loop — derives a gate subset from `git diff`.
capture_goldens       (re)captures goldens. A blessing action, not a check.

# ── measurement (no pass/fail verdict) ──
bench                 perf harness. Reports numbers; asserts nothing.
profile_compiler      per-stage time/allocation profiler.
tmc_census            emits the TMC census that diff_compiler_tmc_parity CONSUMES.
argstamp_parity_probe an investigation probe from the argstamp arc.
w1                    a scratch wasm probe.
